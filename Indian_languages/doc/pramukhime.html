Class: PramukhIME

PramukhIME

PramukhIME is a lightweight, highly customizable javascript library for enabling IME for any HTML element like textbox, textarea, iframe or div. This library requires keyboard plugin in order to type in the language supported by that keyboard plugin. Using 4 lines of code you can enable any Webpage to have PramukhIME on it. As soon as the library is loaded in the Webpage, it creates pramukhIME object which will be used for IME functionality.
<html>
<head>
    <script type="text/javascript" src="js/pramukhime.js"></script>
    <script type="text/javascript" src="js/pramukhindic.js"></script>
</head>
<body>
    <input type="text" id="first_name">
    <script language="javascript" type="text/javascript">
        pramukhIME.addKeyboard(PramukhIndic,"sanskrit"); 
        pramukhIME.enable();
    </script>
</body>
</html>
Features
  • Requires only 4 lines of code to enable PramukhIME on any Web page.
  • Supports multiple keyboard layouts for any language
  • Customizable keyboard shortcut to toggle between currently and last selected language.
  • Provides fine control over chosing HTML element to enable PramukhIME.
  • Supports multiple desktop and mobile browsers.
  • Developer can create his/her own keyboard layout and easily integrate with the library.
  • Once the javascript is downloaded into the browser, it works offline and does not need internet connection
  • Ideal for HTML based mobile application and websites

setToggleKey(key, ctrl, alt)

PRO Version Only
Sets the shortcut key to switch between selected language and last selected language. Default value for toggle key is 120 (F9 key). If the user is typing in a text box(or any PramukhIME enabled HTML element) and presses F9 key, user can immediately start typing in English (given the English is the last selected language). Press F9 key to again type in previously selected language.
Parameters:
Name Type Description
key int Key Code in KeyDown event.
ctrl boolean Hold down Control key for switching between English and selected language?
alt boolean Hold down Alt key for switching between English and selected language?
See:
  • PramukhIME#getToggleKey
Example
// Set the toggle key to Ctrl + F8 so that when user presses Ctrl + F8, it toggles between current and last selected language
pramukhIME.setToggleKey(119, true, false); // 119 = keycode for F8

toggleLanguage()

PRO Version Only
Toggles between selected language and last selected language
See:
  • PramukhIME#getLanguage, PramukhIME#getLastLanguage

output = pramukhIME.getLanguage();

Events

languagechange

PRO Version Only
Language change event
Type:
  • object
Properties:
Name Type Description
language string Newly selected language name
kb string Newly selected keyboard name
lastLanguage string Last selected language
lastKb string Last selected keyboard name
Example
// define language change handler
function languageChangeCallback (e) {
	// We will have following properties available in the argument
	// e.language = 'sanskrit', e.kb = 'pramukhindic', e.lastLanguage = 'english', e.lastKb = 'pramukhime'
}
// Subscribe language change event
pramukhIME.on('languagechange', languageChangeCallback);
// At this moment, we have the English as the default language
// Adding a keyboard and setting the language which would automatically call languageChangeCallback function
pramukhIME.addKeyboard(PramukhIndic,'sanskrit');

settingschange

PRO Version Only
Settings change event
Example
// define settings change handler
function settingsChangeCallback () {
	// Setting change event does not have any argument. It is just for notification.
	// Show/Hide/Modify settings UI based on selected language
}
// Subscribe language change event
pramukhIME.on('settingschange', settingsChangeCallback);
// Add a keyboard and set the selected language
pramukhIME.addKeyboard(PramukhIndic,'sanskrit');
var settings = [{language:'all', kb:'pramukhindic', digitInEnglish: true}];
// As we set the settings, it will call the callback handler "settingsChangeCallback"
pramukhIME.setSettings(settings);
