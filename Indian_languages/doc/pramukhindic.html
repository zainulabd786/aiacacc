Class: PramukhIndic

PramukhIndic

PramukhIndic class contains the phonetic keyboard layout for Indian languages. It can be used with PramukhIME to enable IME functionality for Indian languages on the webpage. It can also be used as a stand alone class to convert English transliteration text into Indian languages. This class supports Assamese, Bengali, Bodo, Dogri, Gujarati, Hindi, Kannada, Kashmiri, Konkani, Maithili, Malayalam, Manipuri, Marathi (Devanagari + Modi), Meitei, Nepali, Odia, Punjabi, Sanskrit, Santali (Ol Chiki), Sindhi, Sora, Tamil and Telugu languages.

Features
  • This library converts the English text into the Unicode and can be used as a transliteration tool
  • Optionally, digits can be kept in English and all other text can be converted into the selected language
  • Powerful rule engine enhances transliteration experience
  • For Hindi, Kannada and Kashmiri, user can set preference for typing Anusvar or Panchamakshar

Defaults
For all languages, DigitInEnglish = false so all the digits will be converted into the selected language
Default language is Hindi

getSettings(languageNamenullable) → {Array.<Settings>}

Gets a settings of the given language. If the language is not provided, it gives the settings of the currently selected language. If you want to get the settings of all the languages, pass 'all' as the parameter. reset()

It resets the state of the object. This function should be called when the input is not continuous (i.e. arrow key pressed, another textbox focused)

resetSettings()

Resets all the settings for all the languages
Example
var kb = new PramukhIndic('hindi');
var settings = [{language:'all', kb: 'pramukhindic', digitInEnglish: true},
		{language:'hindi', kb: 'pramukhindic', advancedRule: 'panchamakshar'} ];
// Sets the Digit in English = true for all languages and Typing Rule = Nasal Consonant for Hindi
kb.setSettings(settings);
// resetting the settings will make the Digit in English = false for all languages and Typing Rule = Anusvar for Hindi
kb.resetSettings();
