About PramukhIME

What is PramukhIME?

PramukhIME is a Javascript library using which any Web developer can enable the visitor to type in Indian language without downloading/installing any special software.

About Developer

Gettting inspiration and blessings from H.D.H. Pramukh Swami Maharaj, Vishal Monpara has dedicated more than 3775 hours from January 2005 till February 2018 to create free utilities for typing in Indian languages.

Supported Browsers

PramukhIME is tested on IE 6+, Chrome, Firefox, Safari and Opera browsers and works as expected.
Even though the script will work in mobile browser Chrome/Firefox, due to lack of standards in Chrome/Firefox browsers, it may not work in rare situation.

Version

This library as a whole is marked as version 4.0.1. This library consists of two individual libraries which has its own version as follows
  • PramukhIME Javascript Library 4.0.0
  • PramukhIndic Keyboard Javascript Library 4.0.1
